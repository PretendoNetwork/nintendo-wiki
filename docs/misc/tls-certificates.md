Many client and server certificates that are used by Nintendo are issued by an internal CA and are valid for a long time. This page lists some of them.

## Nintendo CA

|--------|---------------------|
| `00`   | `Nintendo CA`       |
| `0D`   | `Wii IOP (J) Pro 1` |
| `1A`   | `Wii NWC Prod 1`    |
